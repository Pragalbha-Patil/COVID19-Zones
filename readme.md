A project to display Zones over India using Data published by Govt. of India.

## Installation
Clone the repo &
Use [Composer](https://getcomposer.org/download/) to setup the project.

```cmd
composer install
```
```cmd
cp .env.example .env
```
```cmd
php artisan migrate && php artisan key:generate
```

## Running the project

```cmd
php artisan serve
```
